## Run Locally

To get a local copy up and running follow these simple steps.

### Clone the project

```bash
git clone https://github.com/nyaupane-netra/chat-app
```

### Go to the project directory
```bash
  cd mern-chat-app
```

### Install dependencies

'''bash
  npm install
'''

'''bash
  cd frontend
  npm install
'''

### Start the server
'''bash
npm run start
'''

### Start the Client

'''bash
//open a new terminal
cd frontend
npm start
'''
